# 사용 기술

- React 19.2.4
- Vite 8.0.4
- Tailwind CSS v4.2.2

# 프로젝트 구조

```
src/
  App.jsx              # 메인 앱
  utils/
    analyzer.js        # 단어 추출 및 빈도수 분석
    reader.js          # URL, 파일, 텍스트 파싱
    stopwords.js       # 불용어 필터링
```

# 기능

## 단어 추출
- URL 입력: 웹페이지 메인 컨텐츠에서 단어 추출
- 파일 업로드: 텍스트 파일에서 단어 추출
- 직접 입력: 텍스트 필드에서 단어 추출

## 단어 분석
- 빈도수 계산: 가장 많이 사용된 단어 순위 표시
- 불용어 필터링: 전치사, 부사 등 수식어 자동 제거
- 사전 API: 단어의 발음기호와 뜻 표시

## 저장/불러오기
- JSON 저장: 추출된 데이터 JSON으로 저장
- JSON 불러오기: 이전 데이터 불러오기

# 기획

- URL의 경우 연결 페이지의 메인 컨텐츠만 읽어서 단어 리스트 생성
- 전치사 부사 등의 수식어(꾸밈말)는 빼고 사용된 단어 리스트만 표시
- 표시된 단어는 발음기호와 뜻을 사전 형태로 표시
